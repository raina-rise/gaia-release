Overview

raina-gaia是一款高性能大规模图形渲染工具，包括服务端渲染引擎和客户端渲染引擎两部分。

raina-gaia支持 矩形(rect)、路径(path)、文本(text)、图片(image)或svg 五种基础图元对象的渲染，支持单核渲染和多核渲染模式。

raina-gaia需要用户提供渲染图元的数据信息，根据用户定义的矩形(rect)、路径(path)、文本(text)、图片(image)或svg属性信息，输出对应的渲染图，示例渲染图如下：
